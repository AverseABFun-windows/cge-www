Name: Castle Game Engine
Version: 7.0-alpha.1
Category: graphics
Author: Michalis Kamburelis
E-mail: michalis.kambi@gmail.com
Description:Cross-platform (desktop, mobile, console) 3D and 2D game engine supporting many asset formats (glTF, X3D, Spine...) and using modern Object Pascal.

<p>Features:</p>

<ul>
  <li><p>Use any 3D or 2D software to create your models in various formats: glTF, X3D, VRML, Spine JSON, Collada...</p></li>

  <li><p>Develop cross-platform applications, for desktop (Windows, Linux, macOS, FreeBSD...), mobile (Android, iOS), consoles (Nintendo Switch) and other devices (Raspberry Pi).</p></li>

  <li><p>Visual editor to design games UI and to build applications, powerful command-line build tool under the hood.</p></li>

  <li><p>Optimized rendering with a lot of graphic effects (physically-based rendering, shadows, mirrors, bump mapping, shader effects, gamma correction...).</p></li>

  <li><p>Build and edit the scene graph (X3D) at runtime. Create 3D processing, visualization tools and CAD applications.</p></li>

  <li><p>Extensible system for game objects, with physics, creatures with AI and navmesh, and more.</p></li>

  <li><p>Access numerous services, like in-app purchases and game services on mobile devices.</p></li>

  <li><p>Create cross-platform user-interface with anchors and automatic scaling.</p></li>

  <li><p>Code in modern Object Pascal, an efficient OOP language with cross-platform open-source compiler (FPC), compiled to a native optimized code.</p></li>
</ul>
